## Installation

After forking and cloning the repo:

The app divided in two parts:
* /server/
  `$npm install`
  `$nodemon app.js`
* /client/
  `$npm install`
  `$npm start`


## Tech Stack

The app has been created using
- Koa : [koajs.com](http://koajs.com)
- MongoDB: [https://www.mongodb.com/](http://docs.sequelizejs.com/en/v3/)
